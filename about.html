Technologies utilisées

Ce site web a été créé avec le framework Bootstrap et avec jQuery. MathJax, Function Plot et les plugins jQuery.print et TOC sont utilisés. {{ site.title }} utilise le moteur Jekyll pour générer ses contenus et est hébergé sur Github Pages.

Graphismes

Les icônes PDF, Imprimer, Partager, Correction, Lien et Bacomathiques (ainsi que Approuvé, Livre, Ordinateur, YouTube et Interrogation) proviennent de IconFinder.
