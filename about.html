Technologies utilisées

Ce site web a été créé avec le framework Bootstrap et avec jQuery. MathJax, Function Plot et les plugins jQuery.print et TOC sont utilisés.

Le site WEB est généré à l'aide du moteur Jekyll et est hébergé sur Github Pages.
