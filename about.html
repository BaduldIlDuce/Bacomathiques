Technologies utilisées

Ce site web a été créé avec le framework Bootstrap et avec jQuery. MathJax, Function Plot, Lunr.js, AnchorJS et les plugins jQuery.print, jQuery.lazy, TOC et isInViewport.js sont utilisés. {{ site.title }} utilise le moteur Jekyll pour générer ses contenus et est hébergé sur Github Pages.

Graphismes

Les icônes PDF, Imprimer, Partager, Correction, et Bacomathiques proviennent de IconFinder.
Les autres icônes proviennent de FontAwesome.
